# Easy A Website - Official Repository

Đây là kho lưu trữ mã nguồn chính thức cho ứng dụng **Easy A Website**, được xây dựng trên nền tảng Next.js (nằm trong thư mục `easya_app`) và tích hợp hệ thống tự động hóa CI/CD thông qua GitHub và Vercel.

## Hướng dẫn phát triển
- Thư mục ứng dụng chính: `easya_app`
- Để chạy local:
  ```bash
  cd easya_app
  npm install
  npm run dev
  ```
